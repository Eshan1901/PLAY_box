# 🎮 Playbox :- A Subscription-Based Gaming Platform

**Playbox** is a web application inspired by platforms like **Steam** and **Spotify**, offering a seamless experience for users to **explore, subscribe, and play** a curated selection of games. Built with a modern tech stack and intuitive UI, it simulates a real-world digital game library system.

## 🔗 Live Demo

👉 [Visit Playbox](https://play-box-indol.vercel.app)

## 🧠 About the Project

This is a personal project created to practice front-end development and UI/UX design principles. It showcases how a **game subscription service** could work, allowing users to discover games, explore genres, and visualize their gaming dashboard.

## 🛠️ Tech Stack

- **React.js** – UI Library  
- **JavaScript (ES6+)**  
- **Tailwind CSS** – Utility-first CSS framework  
- **Vercel** – Deployment  
- *(Optional future plans: Firebase/Auth, API integration for game data)*

## ✨ Features

- 🕹️ Browse a collection of games with images, genres, and brief descriptions  
- 🔍 Filter or sort by game type  
- 💾 Add games to “Your Library”  
- 🎨 Clean, modern, responsive UI design  
- 🌓 Dark/light theme ready (optional for expansion)
